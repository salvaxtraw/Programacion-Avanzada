# Análisis de Optimización

## Python

### Versión original
Complejidad aproximada:
O(n²)

Motivo:
- Búsquedas lineales dentro de listas
- Recorridos repetidos de la lista

### Versión optimizada
Complejidad:
O(n)

Motivo:
- Uso de diccionarios (hash table)
- Eliminación de bucles anidados


## C

### Versión original
Complejidad aproximada:
O(N²)

Motivo:
- Comprobación de divisores desde 2 hasta m-1


### Versión optimizada
Complejidad:
O(N√N)

Motivo:
- Solo se prueban divisores hasta sqrt(m)
- Uso de break para detener búsqueda
